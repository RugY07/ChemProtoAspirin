# Chemistry Lab – Aspirin Synthesis

Interactive chemistry lab web app for exploring aspirin synthesis.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local development

**Requirements:** Node.js and npm (or [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) for Node version management).

```sh
# Install dependencies
npm install

# Start the dev server (runs at http://localhost:8080)
npm run dev
```

Other scripts:

- `npm run build` – production build
- `npm run preview` – preview production build locally
- `npm run lint` – run ESLint
- `npm run test` – run tests

## Deploying

Build the app with `npm run build`, then deploy the generated `dist/` folder to any static host (Vercel, Netlify, GitHub Pages, etc.).
